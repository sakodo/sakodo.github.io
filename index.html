A propos

Après des années d'expérience dans l'entrepreneuriat et le management, notamment dans la restauration et chez Orange, j'ai décidé de me réorienter vers un domaine qui m'a toujours passionné : le développement informatique. J'ai démarré en autodidacte avec Python, puis j'ai opté pour des cours à distance chez Studi pour devenir développeur Full Stack. En parallèle, j'ai renforcé mes compétences en présentiel avec la Passerelle, me concentrant sur le backend avec PHP et SQL. Je suis désormais prêt à mettre à profit cette expertise dans un environnement stimulant et innovant.

FRONT

  • HTML (HyperText Markup Language) : Utilisé pour créer la structure de la page web en utilisant des balises.
  • CSS (Cascading Style Sheets) : Utilisé pour styliser la présentation de la page web en définissant les styles, les couleurs, la mise en page, etc.
  • JavaScript : Utilisé pour rendre la page interactive en ajoutant des fonctionnalités dynamiques, des animations et des manipulations du DOM (Document Object Model).

    • SQL (Structured Query Language):

    • Type de base de données : SQL est un système de gestion de base de données relationnelle (SGBDR). Il stocke les données dans des tables liées les unes aux autres par des relations.
    • Structure des données : Les données sont organisées en tables, chaque ligne représentant un enregistrement individuel et chaque colonne représentant un attribut de l'enregistrement.

    Mongobd

    • Type de base de données : MongoDB est un système de gestion de base de données NoSQL (Not Only SQL) orienté document. Il stocke les données sous forme de documents JSON (BSON) dans des collections.
    • Structure des données : Les données sont stockées sous forme de documents flexibles et non structurés. Chaque document peut avoir une structure différente, ce qui permet une flexibilité dans la modélisation des données.
